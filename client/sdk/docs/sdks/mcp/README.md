# Mcp
(*mcp*)

## Overview

Model Context Protocol server hosting.

### Available Operations

* [mcpNumberServe](#mcpnumberserve) - serve mcp

## mcpNumberServe

MCP server endpoint for a toolset.

### Example Usage

```typescript
import { Gram } from "@gram/client";

const gram = new Gram();

async function run() {
  const result = await gram.mcp.mcpNumberServe({
    project: "<value>",
    toolset: "<value>",
    environment: "<value>",
  });

  // Handle the result
  console.log(result);
}

run();
```
